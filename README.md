# Cat Health Tracker

A comprehensive Streamlit application for tracking the health and care tasks of multiple cats.

## Features

- 🐱 Individual cat health tracking for Haku, Kuro, and Sonic
- 📝 Daily health entries (water, food, litter box usage, mood, medication)
- 📋 Task management with daily, weekly, monthly, and quarterly scheduling
- 🤖 AI-powered health analysis and chat assistant
- 🎯 Comprehensive dashboard with insights and trends
- 💾 Persistent data storage
- 📥 Excel export functionality
